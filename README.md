# Fundraisers
Fundraiser App is built with:

- Vue
- Vuex
- Axios
- Tailwindcss
- GoFundRaise.com API (api.gofundraise.com.au)

## Installing

Install the dependencies and devDependencies and start the server.

```sh
$ cd fundraisers
$ npm install
$ npm run serve
```

## Deployment

```sh
$ npm run build
```
