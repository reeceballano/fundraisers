# Fundraisers
Fundraiser App is built with:

- Vue
- Vuex
- Axios
- Tailwindcss
- GoFundRaise.com API (api.gofundraise.com.au)

## Features

### User Can:
- Search a specific campaign ID
- Select Individual or Team
- Select how many search results to be displayed
- Switch to Light or Dark mode theme 

## Installing

Install the dependencies and devDependencies and start the server.

```sh
$ cd fundraisers
$ npm install
$ npm run serve
```

## Deployment

```sh
$ npm run build
```

## Unit Testing
```sh
$ npm run test:unit
```
